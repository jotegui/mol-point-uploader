Welcome to the observation loader, blah blah...

Steps for the correct upload of observation data:

  1. Download the spreadsheet template
  2. Fill it out. You can find some information about each field here, if you need it
  3. Save the spreadsheet as a csv file
  4. Upload the spreadsheet with the button below

You can also upload the data in your own format, as long as it has the following fields (or default values for them):

  • Scientific Name
  • Latitude and Longitude, preferrably in decimal degrees
  • Observation Date
  • Collector's name
